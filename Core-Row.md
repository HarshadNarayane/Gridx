The `Row` object is a core part of GridX.  It models a Row in the table.  It has a number of methods including:

* `cell(...)`
* `cells(...)`
* `data()`
* `deselect()`
* `index()`
* `isSelected()`
* `item()`
* `moveTo(...)`
* `node()`
* `parent()`
* `rawData()`
* `select()`
* `setRawData(rawData)`
* `visualIndex()`
